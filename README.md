# imprint

Convert images and videos to string representations.    
![text flower](/img/flower_out.png)

## Installation
```
$ git clone https://github.com/jacoblurye/imprint.git
$ cd imprint
$ python setup.py install
```

## Usage
### Via the CLI
Call the `imprint` command on any supported image or video file format.
```
$ imprint [options] file
```

>>> imprinter = ImgPrinter(symbols)
>>> imprinter('img/flower.jpg', max_width=75)
